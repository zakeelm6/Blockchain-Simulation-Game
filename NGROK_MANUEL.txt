# GUIDE MANUEL NGROK - PAS A PAS

## ETAPE 1 : Demarrer le Backend

Ouvrez PowerShell Terminal 1 :

```powershell
cd D:\Blockchain-Simulation-Game\backend
npm start
```

Laissez cette fenetre OUVERTE.

---

## ETAPE 2 : Tunnel ngrok Backend

Ouvrez PowerShell Terminal 2 :

```powershell
cd D:\Blockchain-Simulation-Game
.\ngrok.exe http 4000
```

Vous verrez quelque chose comme :

```
Forwarding   https://abc123.ngrok-free.app -> http://localhost:4000
```

COPIEZ cette URL (exemple: https://abc123.ngrok-free.app)

Laissez cette fenetre OUVERTE.

---

## ETAPE 3 : Configurer le Frontend

Ouvrez PowerShell Terminal 3 :

```powershell
cd D:\Blockchain-Simulation-Game\frontend

# Remplacez URL_DU_BACKEND par l'URL copiee a l'etape 2
echo VITE_API_URL=https://abc123.ngrok-free.app > .env.local

npm run dev
```

Laissez cette fenetre OUVERTE.

---

## ETAPE 4 : Tunnel ngrok Frontend

Ouvrez PowerShell Terminal 4 :

```powershell
cd D:\Blockchain-Simulation-Game
.\ngrok.exe http 5173
```

Vous verrez quelque chose comme :

```
Forwarding   https://xyz789.ngrok-free.app -> http://localhost:5173
```

C'EST CE LIEN QUE VOUS PARTAGEZ AUX ETUDIANTS !

Laissez cette fenetre OUVERTE.

---

---

## IMPORTANT

- Gardez les 4 fenetres PowerShell ouvertes
- Ne fermez pas votre PC
- Session ngrok : 2 heures max
- Pour arreter : Ctrl+C dans chaque fenetre

---

## RESUME DES COMMANDES

Terminal 1 : cd D:\Blockchain-Simulation-Game\backend && npm start
Terminal 2 : cd D:\Blockchain-Simulation-Game && .\ngrok.exe http 4000
Terminal 3 : cd D:\Blockchain-Simulation-Game\frontend && npm run dev
Terminal 4 : cd D:\Blockchain-Simulation-Game && .\ngrok.exe http 5173
